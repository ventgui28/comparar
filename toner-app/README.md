# Comparação de Tinteiros (Toner App)

Aplicação React para comparar preços de consumíveis a partir de tabelas Excel.

## Funcionalidades
- Importação de ficheiros Excel com mapeamento flexível de colunas.
- Normalização inteligente de referências para agrupamento automático.
- Histórico de preços local (IndexedDB) para produtos favoritos.
- Visualização de tendências e melhores ofertas.

## Tecnologias
- React + Vite
- IndexedDB para armazenamento local
- Lucide React para ícones
- XLSX para processamento de Excel

## Desenvolvimento

### Instalação
```bash
cd toner-app
npm install
```

### Executar
```bash
npm run dev
```

### Testes
```bash
npm test
```
